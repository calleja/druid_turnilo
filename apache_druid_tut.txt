membership_ard steps:
1. Extract data (from query) from mysql db: run query to export table to csv (do this from a terminal window in the docker container)

*** can the mysql instance access other directories? It's those that I will bind-mount to the Druid container
2. add a header if one is not present (inspect with command to see first row: cat membership_viz.csv | head -n 1)
sed -i '' '1s/^/calendar_date,active_cnts,temporary_inactivte,revoked,general_leave,winbacks\n/' membership_viz.csv
3. copy the csv into a druid container via my local (essentially must route data - via a hard copy - through my physical local filesystem - can't do it in Docker):
3.1 MySQL container → your Mac (current directory)
> docker cp mysql-cont:/membership_viz.csv ./membership_viz.csv
3.2 your Mac → middlemanager's shared volume
> docker cp ./membership_viz.csv middlemanager:/opt/shared/membership_viz.csv
4. run query I've stored in /druid_tut/load_membership_viz.sql
